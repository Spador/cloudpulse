## Typography

- **Primary font**: Inter (Google Fonts)
- **Monospace font**: JetBrains Mono (for instance IDs, function names)
- **Headings**: `font-bold` or `font-semibold`
- **Body**: `font-normal`

## Component Patterns

- Cards: `bg-white rounded-xl border border-gray-200 p-6`
- Primary button: `bg-blue-600 text-white rounded-lg px-4 py-2`
- Badge (critical): `bg-red-100 text-red-800`
- Badge (high): `bg-orange-100 text-orange-800`
- Badge (medium): `bg-yellow-100 text-yellow-800`
